Algorithmic Game Theory

Computing equilibria and learning strategies in multi-agent strategic settings.

Many real-world systems involve strategic interactions — defenders and attackers, planners and responders. We develop scalable methods for computing and learning equilibria in these settings.

Key directions:

  • End-to-end gradient descent for Stackelberg games
  • Deep fictitious play for continuous action spaces
  • Security games and green security patrol scheduling
  • Learning adversary models from data

Selected publications:

(Wang et al., 2022; Niu et al., 2025; Kamra et al., 2019; Cooney et al., 2019; Wang et al., 2020)
